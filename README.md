# Login and authentication page with an dynamic background using spline.
***
## Tech stack used 
- React
- Node
- Express
- mongoDB
- Tailwind CSS

## For running this specific project use
- A mongoDB connection link as MONGO_URI in dbSetup
- A VITE_SERVER_URL enviroment as the endpoint to the backend.
